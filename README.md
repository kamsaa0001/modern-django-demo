# Modern Django Website 🚀

## 📌 Description
This is a Django-based web application developed to demonstrate web development skills.

## ⚙️ Features
- User authentication (Login/Signup)
- Dynamic pages
- Admin panel
- Database integration

## 🛠️ Tech Stack
- Python
- Django
- HTML, CSS

## 🚀 How to Run
1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Run server:
   python manage.py runserver
